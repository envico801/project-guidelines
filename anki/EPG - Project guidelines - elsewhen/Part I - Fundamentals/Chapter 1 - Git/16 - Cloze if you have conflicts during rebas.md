###  If you have conflicts during rebasing, you should {{c1::resolve them}} and then continue the rebase using {{c2::git rebase --continue}}.  

========== Answer ==========  

(Cloze) To resolve conflicts, use:

```sh
git add <file1> <file2> ...
git rebase --continue
```
